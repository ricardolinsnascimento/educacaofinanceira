O que foi feito em cada disciplina?

Matemática ->

  • Completar a planilha de custos de energia elétrica dos seguintes aparelhos domésticos:
  • chuveiro;
  • computador;
  • ar condicionado;
  • geladeira;
  • lâmpada.

Língua portuguesa ->

  • Produzir um texto relacionado a planilha de Matemática sobre quatro aparelhos citados acima:
  • chuveiro;
  • computador;
  • geladeira;
  • lâmpada.

Ferramentas de desenho ->

  • Criar o cenário de uma casa, apresentando os cômodos e aparelhos domésticos.
