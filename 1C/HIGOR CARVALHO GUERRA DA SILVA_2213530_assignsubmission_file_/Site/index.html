Esse trabalho foi feito sobre o consumo de energia em que cada pessoa consomem durante um mês, foi pego cinco eletrodomesticos para saber quanto cada um consomemem.Com isso em que cada matéria ficou responsável por uma coisa.O professor de Matemática o Sonivaldo Ruzzene Beltrame deu uma tabela para fazermos os cálculos de cada eletrodoméstico, com isso a professora de Português a Mariza Perobelli deu uma folha para fazermos uma redação com as informações que fizemos na aula de Matemática.E depois com a professora de Ferramentas de Desenho a Fabricia Ferreira de Souza usamos uma ferramenta de desenho, em desenhamos uma casa como ilustração da nossa com cada comôdo.E por último com o professor de Front-End o Ricardo Augusto Lins do Nascimento.usamos todas essas informações para criar um site.
