Trabalho Multidisciplinar-Educação Finâncira

Este trabalho feito por Antonio Mendes Neto é um trabalho Multidisciplinar de Educação Finânceira.Este trabalho foi feito em quatro matérias diferentes.Em matemática foi feito uma planilha para colocar os dados de alguns aparelhos e resolver as equações.Em lingua portuguesa foi feito um texto sobre esta planiha e comentando sobre alguns itens.Em ferramenta de desenho foi produzido um desenho para retratar uma casa com alguns cômodos.E por ultimo em front-end foi desenvolvido este site contendo todas as atividades das matérias listadas.

Os professores de cada matéria são:

  • Lingua portuguesa: professora Mariza Perobelli
  • matemática: professor Sonivaldo Beltrame
  • Ferramentas de desenho: professora Fabricia Ferreira
  • Front-end: professor: Ricardo Augusto
