Educação Financeira

O trabalho multidiciplinar tem como tema: educação finaceira ,esse trabalho é para que os alunos veja a importacia para a economia financeira a economia de energia, por gasto de energia á também desperdicio de água.

Com o professor Sonivaldo de Matematica agente fez os cálculos para saber a quantidade de energia eo valor que agente gasto mensal.

Com a professora Mariza agente produziu um texto usando os dados que fizemos em matematica.

Já com o professor Thiago de feramenta de desenho fizemos um desenho da nossa casa.
