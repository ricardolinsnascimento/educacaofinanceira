Educação Financeira

O objetivo deste trabalho é conscientizar os estudantes sobre seu consumo individual de energia. Cada máteria teve uma participação importante para a conclusão do projeto. O professor Sonivaldo, responsável pela disciplina de Matemática, pediu a criação de uma tabela com informações do consumo de cada aluno, como potência em Watts de cada produto, tempo de uso, quantidade de dias de uso, consumo mensal e valor do kWh. A professora Mariza, responsável pela disciplina de Português, pediu a produção de um texto com base nas informações da tabela entregue pelo professor Sonivaldo. O professor Thiago, responsável pela disciplina de Ferramenta de Desenho, ajudou a criar uma planta da residência dos alunos. O professor Ricardo deu tarefa de criarmos este site para apresentar o trabalho completo.
