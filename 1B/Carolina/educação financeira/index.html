Trabalho Multidisciplinar - Educação Financeira

O trabalho de Educação Financeira, Multidisciplinar, foram feitos com os alunos do 1º semestre do CTI Técnico em Informática para Internet, com o intuito de trazer interesse aos estudantes. E também para termos consciência do quanto gastamos e diminuir o custo.

As matérias utilizadas para esse trabalho foram:

Matemática

O professor Sonivaldo Beltrame aplicou um quadro para preenchermos com os dados que obtemos sobre o consumo de energia elétrica de nossa residência. Foi dado 7 itens para completarmos.

  • Pência (watts);
  • Tempo (min);
  • Consumo Diário (kw/h);
  • Quantidade de dias;
  • Consumo mensal (kW/h);
  • Valor kw/h (R$);
  • Consumo Mensal (R$).

Juntamente com os 7 itens, foram dados 4 aparelhos para fazermos as pesquisa dos dados:

  • Chuveiro;
  • Geladeira;
  • Televisor;
  • Lampâda.

LPLB (Lingua Portuguesa e Literatura Brasileira)

A professora Mariza Perobelli aplicou com as turmas a produção de texto, onde tinhamos que produzir de uma forma específica, descrevendo quadro de Matemática, no qual precisavamos compreender o propósito do trabalho.

Produzimos os textos com descrevendo os números, os itens e as especificações.

Ferramentas de Desenho

O professor Tiago Gomes introduziu as turmas a fazerem um mapa da casa ou dos cômodos, onde os 4 aparelhos estão instalados.

Esse é um exemplo que o próprio professor sugeriu aos alunos.

Desenvolvimento Front End

O professor Ricardo Nascimento passou para as turmas um trabalho, no qual tinhamos que fazer um site.

Nesse site, deveriamos falar sobre as especificações de cada matéria utilizada, com imagens, critérios, links externos e internos.
