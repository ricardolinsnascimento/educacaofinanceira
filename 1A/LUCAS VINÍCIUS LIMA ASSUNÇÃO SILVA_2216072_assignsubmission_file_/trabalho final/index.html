Trabalho Multidiciplinar-Eucação Financeira

Oque foi Feito?

Nese trabalho, em Matemática, a qual a professora da disciplina é a Cristiane Bender foram pegos os dados de consumo de energia de 5 eletrodoméstico de casa (Ar-condicionado, Televisão, Cuveiro, Lâmpada e Geladeira).

Já em Português, A professora Mariza propôs que os alunos fizessem um texto descrevendo os dados dessa pesquisa. No entanto, só foram descritos 4 eletrodomesticos.

Em Ferramentas de Desenhos, o Professor Tiago pediu para que seja feita a planta de uma casa com os móveis e os cômodos pertencentes á ela.

E em Front-End 1, o Professor Ricardo pediu para os alunos desenvolver um site para inserirmos os relatados a cima.

Este vídeo do Youtube é uma aula sobre Educação Financeira que você pode assistir clicando nele.
