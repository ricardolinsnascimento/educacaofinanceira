Trabalho Multidisciplinar - Educação Financeira

O que é o Trabalho?

O trabalho Multidisciplinar - Educação Financeira foi um trabalho feito com um conjunto de 3 máterias que buscou melhorar o entendimento sobre os gastos com energia elétrica dentro das casas dos estudantes e incentivar a conscientização dos estudantes participantes.


Matérias e Professores envolvidos

  • Português = Mariza
  • Matemática = Cristiane
  • Ferramentas de Desenho = Tiago
  • Front-End = Ricardo
